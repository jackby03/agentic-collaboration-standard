# ACS Governance

## Current status

ACS is currently maintained by [@jackby03](https://github.com/jackby03).

## Decision process

- **Non-breaking clarifications**: maintainer approval
- **New fields or file types**: open issue → 2-week comment period → maintainer decision
- **Breaking changes to v1**: requires RFC + 30-day comment period

## Future governance

As adoption grows, ACS aims to move to a neutral foundation (similar to AGENTS.md → Linux Foundation / Agentic AI Foundation).
