This file lists the initial maintainers and contacts for the Agentic Collaboration Standard (ACS).

Community maintainers (volunteers)

Maintainer responsibilities
- Triage issues and PRs in a timely manner
- Review and merge non-breaking documentation fixes
- Coordinate major changes via RFCs and maintain governance process
- Act as contacts for outreach to AAIF and other projects

Process to add maintainers
- Submit a PR adding the maintainer entry and a short justification.
- Current maintainers must approve the addition (maintainer +1).
