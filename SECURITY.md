# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x (CLI) | ✅ |
| 0.1.x (VS Code) | ✅ |
| < 1.0.0 | ❌ |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Use GitHub's [private vulnerability reporting](https://github.com/jackby03/agentic-collaboration-standard/security/advisories/new) to report issues confidentially.

Alternatively, email: **jackby03@outlook.com**

### What to include

- Description of the vulnerability
- Steps to reproduce
- Affected component (CLI, VS Code extension, spec)
- Potential impact

### Response timeline

- **Acknowledgement:** within 48 hours
- **Status update:** within 7 days
- **Fix / advisory:** within 30 days for confirmed vulnerabilities

Reporters who responsibly disclose vulnerabilities will be credited in the security advisory unless they prefer to remain anonymous.
