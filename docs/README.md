# Project Documentation

This directory now contains project documentation only.

The main future-facing work is tracked in the proposal docs under `proposals/` and reflects community feedback such as [issue #9](https://github.com/jackby03/agentic-collaboration-standard/issues/9), which asked whether ACS should pursue AAIF / Linux Foundation-style standardization and stronger cross-tool `.agents/` interoperability. That same proposal track is also where adjacent standards such as DESIGN.md / getdesign.md should be evaluated for selective adoption.

## Structure

- `guides/` — user and contributor documentation
- `proposals/` — drafts, interoperability notes, and exploratory design work

## Website assets

The static website and branding assets live in [`../website/`](../website/) so they do not mix with the project documentation surface.
