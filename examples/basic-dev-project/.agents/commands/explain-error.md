## Task

Given the error message, explain:

1. What caused the error (plain language, no jargon)
2. Where in the code it likely originated
3. How to fix it (step by step)
4. How to prevent it in the future

## Style
- Simple language, assume intermediate developer
- Show code examples for the fix
- Max 200 words total
