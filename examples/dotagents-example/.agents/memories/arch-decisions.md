# Architecture Decisions

## Keep the source of truth portable

The project prefers formats that can be translated across agents and tools instead of storing critical guidance in a single vendor-specific file.

## Validate before standardizing

Experimental interoperability structures should be documented and tested before being proposed as normative ACS behavior.
