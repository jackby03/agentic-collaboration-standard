# ACS Reference Implementations

Lightweight parser and validator implementations for ACS.

These are **reference implementations** — designed for clarity and correctness, not production performance.

## Python
- [acs_parser.py](python/acs_parser.py) — Discover and parse ACS files
- [acs_validator.py](python/acs_validator.py) — Validate against schemas

## TypeScript
- [parser.ts](typescript/parser.ts) — Discover and parse ACS files
- [validator.ts](typescript/validator.ts) — Validate against schemas
